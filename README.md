# Context Registry Test Fixture

This repository is intentionally small and registry-focused. It contains
Markdown artifacts under the node-type directories that the authz context
registry currently classifies:

- `skills/`
- `prompts/`
- `tools/`

Each artifact uses YAML frontmatter to exercise registry metadata and
relationships:

- `uses`
- `requires`
- `implements`
- `derived_from`
